# 온라인으로 믹싱마스터링 의뢰하는 방법 — 처음이라 모르겠는 분들 보세요

지방에 계시거나 직접 방문하기 어려운 분들이 많이 물어보시는 것 중 하나가 바로 이거예요.

"온라인으로 **믹싱마스터링** 의뢰 어떻게 해요?"

파일만 보내면 되는 거 맞냐고요? 맞아요. 근데 어떤 파일을 어떻게 보내야 하는지 몰라서 막히는 분들이 많더라고요. 오늘 전 과정을 정리해드릴게요.

---

## 온라인 의뢰 전체 흐름

> 의뢰 문의 → 서비스 확정 → 파일 준비 → 파일 전송 → 작업 진행 → 수정 피드백 → 최종 납품

간단해요. 제일 중요한 건 **파일 준비**예요.

---

## 어떤 파일을 보내야 해요?

**마스터링 단독 의뢰라면:**
- 완성된 스테레오 믹스 파일 (WAV, 44.1kHz / 48kHz, 24bit 권장)
- 피크가 -3dBFS~-6dBFS 이하인 파일 (너무 크게 믹스되지 않은 것)
- "마스터링 체인 없이 보내주세요"라는 말 많이 하는데, 이미 리미터가 잔뜩 걸린 파일은 마스터링이 제대로 안 돼요

**투트랙 믹싱+마스터링이라면:**
- 악기 반주 스테레오 파일 (MR) + 보컬 WAV 파일
- 이렇게 두 개 파일만 있으면 시작할 수 있어요

**멀티트랙 믹싱+마스터링이라면:**
- 각 트랙을 개별 WAV로 export한 파일 묶음
- 드럼, 베이스, 기타, 신스, 보컬 등 각각의 트랙
- DAW 프로젝트 파일의 시작 지점을 맞춰서 보내주셔야 해요 (흔히 "트랙 정렬"이라고 해요)

## 파일 전송 방법

- **WeTransfer** (무료, 최대 2GB)
- **구글 드라이브** (공유 링크)
- **드롭박스**
- **네이버 클라우드**

카카오 채널로 링크만 보내주시면 됩니다.

## 레퍼런스 곡도 같이 보내주세요

"어떤 방향으로 해주세요"를 말로 설명하는 것보다 레퍼런스 곡을 함께 보내주시면 훨씬 빨리 방향이 맞아요.

"이 곡처럼 따뜻하게", "이 곡처럼 선명하게" — 이런 방식으로요. 유튜브 링크나 스포티파이 링크로 보내주셔도 돼요.

---

## 수정은 어떻게 해요?

작업 후 파일을 먼저 보내드리고, 피드백을 받아서 수정해요. 카카오 채널로 "이 부분이 좀 너무 밝은 것 같아요", "보컬이 조금 묻히는 것 같아요" 같이 말씀해주시면 돼요.

MADEWELL MUSIC은 픽스 전까지 수정 횟수에 제한을 두지 않아요. 방향이 맞을 때까지 같이 작업합니다.
