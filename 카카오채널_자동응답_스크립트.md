## 설정 방법 (카카오 채널 관리자센터)

1. [카카오 채널 관리자센터](https://business.kakao.com) 접속
2. 채널 선택 → **스마트채팅** 메뉴 클릭
3. **자동응답** 탭 → **+추가** 버튼
4. 키워드 입력 (예: `가격, 얼마, 비용`) + 응답 메시지 붙여넣기
5. **인사말**은 채널홈 → **채널 관리** → **인사말 설정**에서 별도 입력

> ⚠️ 키워드는 쉼표로 구분해서 여러 개 등록 가능
> ⚠️ 메시지 길이 제한: 1,000자 이내
