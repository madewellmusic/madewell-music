---

## 전체 워크플로우 (하루 30분 목표)

```
[정균] 촬영 (10분)
    ↓
[Claude] 스크립트 · 캡션 · 해시태그 생성 (5분)
    ↓
[CapCut] 자막 · 편집 (10분)
    ↓
[인스타] 릴스 업로드 → 유튜브 쇼츠 · 틱톡 동시 배포 (5분)
```

---

## Step 1 — 촬영 가이드

| 항목 | 기준 |
|------|------|
| 비율 | **9:16 세로** (필수) |
| 길이 | **15~60초** (30초 권장) |
| 해상도 | 1080×1920 이상 |
| 도구 | 스마트폰 + 삼각대 or 스튜디오 거치대 |
| 포인트 | **처음 3초에 훅** — 장비 클로즈업 / 반응 / 강렬한 사운드 |

### 촬영 유형별 팁
- **장비 소개**: NEVE 1073 / SSL FUSION 버튼 조작 클로즈업 → 소리 before/after
- **믹싱 팁**: 화면 캡처 + 손 조작 화면 인서트컷
- **커리어 비하인드**: 앨범 자켓 / 무대 영상 + 정균 해설 보이스오버
- **카운트다운**: 달력 / D-카운트 텍스트 오버레이

---

## Step 2 — Claude 프롬프트 템플릿

아래 템플릿 복사해서 Claude에게 붙여넣기:

```
오늘 릴스 주제: [주제 한 줄]
영상 내용: [찍은 내용 2~3줄 설명]
타겟: [예: 인디 아티스트 / K-POP 팬 / 음악 배우고 싶은 사람]

아래 3가지 생성해줘:
1. 자막 스크립트 (30초 기준, 말하는 속도 고려)
2. 인스타 캡션 (한국어 + 영어 2버전, 각 150자 내외)
3. 해시태그 30개 (한국어 15 + 영어 15)
```

---

## Step 3 — CapCut 편집 세팅

1. **자동 캡션** 기능 ON (CapCut AI 자막)
2. 폰트: 굵은 흰색 + 검정 테두리 (가독성 최우선)
3. **첫 프레임**에 훅 텍스트 오버레이 (예: "이 장비 하나로 K-POP 마스터링합니다")
4. BGM: 본인 작업곡 10~15초 클립 (저작권 안전 + 자기 브랜딩)
5. 아웃트로 3초: 로고 + "@madewellmusic_official"

---

## Step 4 — 플랫폼별 배포

| 플랫폼 | 방법 | 추가 세팅 |
|--------|------|-----------|
| **인스타 릴스** | 직접 업로드 (원본) | 커버 이미지 별도 설정, 피드 공유 ON |
| **유튜브 쇼츠** | 동일 파일 업로드 | 제목 = 캡션 첫 문장, #Shorts 태그 필수 |
| **틱톡** | 동일 파일 업로드 | 인스타 워터마크 제거 후 업로드 (CapCut 저장본 사용) |

> **업로드 최적 시간**: 오전 7~9시 or 저녁 8~10시 (한국 기준)
